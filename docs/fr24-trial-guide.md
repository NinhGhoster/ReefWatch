# Flightradar24 Business Trial Guide

## Steps to Get Satellite ADS-B Data

1. **Go to:** https://www.flightradar24.com/premium
2. **Sign up for free trial** (Business tier has satellite coverage)
3. **Search for each Spratly airport code:**
   - RCSP — Taiping Island (Taiwan)
   - RPPN — Thitu Island / Pag-asa (Philippines)
   - LAC — Layang-Layang (Malaysia)
   - AG 4553 — Fiery Cross Reef (China)

4. **For airports without ICAO codes**, search by coordinates:
   - Fiery Cross Reef: 9.53°N, 112.88°E
   - Subi Reef: 10.88°N, 114.07°E
   - Mischief Reef: 9.90°N, 115.52°E
   - Trường Sa Island: 8.64°N, 111.92°E

5. **Check playback/history mode** — see if any flights appeared in the last 30 days
6. **Enable satellite layer** in the map settings to see Aireon coverage

## What to Look For

- Military transport (Y-8, Y-20, C-130) doing resupply runs
- Patrol aircraft (KQ-200, P-8A) on surveillance missions
- Helicopters between islands
- Any aircraft with transponders on (even briefly)
